# 📊 Power BI Sales Dashboard Project (End-to-End Data Analysis)

## 🚀 Project Overview

This project demonstrates an end-to-end data analytics workflow using Power BI. The main objective of this project is to analyze sales data and generate meaningful business insights through interactive dashboards and visualizations.

The project covers data cleaning, transformation, data modeling, and dashboard creation to help stakeholders make data-driven decisions.

## 🎯 Objectives

* Analyze sales performance across different regions, products, and time periods
* Identify key trends and patterns in revenue and profit
* Create interactive dashboards for better visualization
* Enable decision-making using data insights

## 🛠️ Tools & Technologies Used

* Power BI
* Microsoft Excel / CSV Dataset
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## 📂 Project Workflow

### 1. Data Collection

* Imported raw sales dataset (Excel/CSV) into Power BI

### 2. Data Cleaning & Transformation

* Removed duplicates and null values
* Changed data types
* Used Power Query Editor for transformation

### 3. Data Modeling

* Created relationships between tables
* Applied star schema (if multiple tables used)

### 4. DAX Calculations

* Created measures like:

  * Total Sales
  * Total Profit
  * Profit Margin
  * Year-to-Date (YTD) Sales

### 5. Dashboard Creation

* Built interactive dashboards using:

  * Bar charts
  * Line charts
  * Pie charts
  * KPI cards
  * Slicers (filters)

---

## 📊 Key Insights

* Identified top-performing products and regions
* Analyzed monthly and yearly sales trends
* Found high-profit and low-profit segments
* Highlighted business growth opportunities
